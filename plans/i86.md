# Add Missing Templates to Demo Presentation

The demo presentation at `demos/demo/` currently showcases 19 of 34 available templates. The following 15 templates have no demo slide:

- `AgendaSlide`
- `BigNumberSlide`
- `ClosingSlide`
- `IconGridSlide`
- `MatrixSlide`
- `ProblemSolutionSlide`
- `ProcessSlide`
- `QuoteSlide`
- `RoadmapSlide`
- `SectionDividerSlide`
- `TeamSlide`
- `TestimonialSlide`
- `TimelineSlide`
- `TableOfContentsSlide`
- `TwoColumnSlide`

Each task below creates the missing slide files and registers them in `demos/demo/deck.ts`. Use the corresponding `src/templates/[name]/example.tsx` as the reference for props. Follow the same patterns as existing demo slides (default export, `.meta` attached, `SectionTitle` for headers). After all slides in a task are written, run `npm run build` to fix any TypeScript errors, then `npm run export -- --deck=demo --format=png` and review every PNG before declaring the task done.

## Task 1 — Presentational / single-concept templates

**New files** (all under `demos/demo/`):

| File | Template |
|---|---|
| `quote.tsx` | `QuoteSlide` |
| `bignumber.tsx` | `BigNumberSlide` |
| `closing.tsx` | `ClosingSlide` |
| `sectiondivider.tsx` | `SectionDividerSlide` |
| `agenda.tsx` | `AgendaSlide` |
| `toc.tsx` | `TableOfContentsSlide` |
| `icongrid.tsx` | `IconGridSlide` |
| `process.tsx` | `ProcessSlide` |

**Guidance per slide:**

- **`quote.tsx`** — A short memorable quote (e.g. something about design or software). Use `QuoteSlide` with `quote`, `attribution`, and `role` props.
- **`bignumber.tsx`** — Pick a striking stat (e.g. "34 templates"). Use `BigNumberSlide` with `value`, `label`, `context`, and `trend="up"`.
- **`closing.tsx`** — Use `ClosingSlide` with a `cta`, `headline`, `subtitle`, and a plausible `contact` block.
- **`sectiondivider.tsx`** — Use `SectionDividerSlide` with `eyebrow`, `title`, `subtitle`, and a `backgroundColor`.
- **`agenda.tsx`** — Use `AgendaSlide` with a `SectionTitle` header and 4–5 agenda items with optional `time` fields.
- **`toc.tsx`** — Use `TableOfContentsSlide` with a `SectionTitle` header, `columns={3}`, and 6 section items each with a Lucide icon and subtitle.
- **`icongrid.tsx`** — Use `IconGridSlide` with a `SectionTitle` header, `columns={3}`, and 6 feature/capability items, each with a Lucide icon, label, and description.
- **`process.tsx`** — Use `ProcessSlide` with a `SectionTitle` header, `direction="horizontal"`, and 4 steps.

**`demos/demo/deck.ts` changes:** import and append all 8 new slides into the `slides` array, inserting them before `TheEndDemo` at the end.

- [ ] `deck.ts` updated
- [ ] `npm run build` passes
- [ ] `npm run export -- --deck=demo --format=png` produces correct PNGs for all new slides

## Task 2 — Structured-data / multi-person templates

**New files** (all under `demos/demo/`):

| File | Template |
|---|---|
| `timeline.tsx` | `TimelineSlide` |
| `matrix.tsx` | `MatrixSlide` |
| `roadmap.tsx` | `RoadmapSlide` |
| `twocolumn.tsx` | `TwoColumnSlide` |
| `problemsolution.tsx` | `ProblemSolutionSlide` |
| `team.tsx` | `TeamSlide` |
| `testimonial.tsx` | `TestimonialSlide` |

**Guidance per slide:**

- **`timeline.tsx`** — Use `TimelineSlide` with `direction="horizontal"`, a `SectionTitle` header, and 4–5 milestones with `date`, `label`, `description`, and one `highlight: true` item.
- **`matrix.tsx`** — Use `MatrixSlide` with a `SectionTitle` header showing a 2×2 priority matrix (e.g. feature prioritization). Provide `xAxis`, `yAxis`, and all four quadrant props; set `highlight: true` on one quadrant.
- **`roadmap.tsx`** — Use `RoadmapSlide` with a `SectionTitle` header, 4 quarterly phases, and 3 rows of work-stream items mixing `done`, `in-progress`, and `planned` statuses. Include at least one item with `span: 2`.
- **`twocolumn.tsx`** — Use `TwoColumnSlide` with a `SectionTitle` header, `ratio="50/50"`, and two columns each with a `title` and 3 string `items`.
- **`problemsolution.tsx`** — Use `ProblemSolutionSlide` with a `SectionTitle` header and 3 problem points paired with 3 solution points.
- **`team.tsx`** — Use `TeamSlide` with a `SectionTitle` header and 4 team members (name, role; no real photos needed — initials fallback is fine).
- **`testimonial.tsx`** — Use `TestimonialSlide` with a `quote`, `author`, `role`, `company`, and `rating={5}`.

**`demos/demo/deck.ts` changes:** import and append all 7 new slides into the `slides` array, inserting them before `TheEndDemo`.
